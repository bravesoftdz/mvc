# Simple MVC Pattern

## Installation
+ run the script ```"db.sql"``` in the folder migration
+ run command ``` composer install --optimize-autoloader```

## Main features
+ mvc folder structure 
+ request routing
+ pdo database connection
+ service provider (Session,Post,Get)
+ error processing page
+ config
+ namespace psr-4
+ layout
+ bootstrap
+ jquery

## In the future
+ support migration
+ template language
